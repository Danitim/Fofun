# Fofun
**Fofun (Фофан)** — исследовательско-инженерный проект по созданию нейросетевой системы, генерирующей шутки и каламбуры **на русском языке**, с опорой на **формально описанные лингвистические и математические теории юмора**.  
Ключевая идея проекта — не просто “обучить модель шутить”, а **напрямую управлять механизмами юмора** через формальные правила, заимствованные из качественных научных исследований, без ручной разметки или ручного кодирования правил для русского языка.
